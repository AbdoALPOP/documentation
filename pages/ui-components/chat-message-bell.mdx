# Chat Message Bell UI component

{/* TODO This component should be renamed to ChatMessageBell */}

The `<ChatMessageBell />` component is typically used in the head offer your application. It informs the user about 
unread messages in the chat.

[See this component on Storybook]()

![Chat Message Bell UI component](/chat-message-bell.png)

## Integration
To show the bell component, you can copy this snippet in your code:

```tsx
<ChatProvider>
 <ChatNotificationBell showZero />
</ChatProvider>
```
