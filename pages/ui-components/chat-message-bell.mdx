# Chat Message Bell UI component

## Introduction

The `<ChatMessageBell />` component is typically used in the header off your application. It informs the user about
unread messages in the chat. Please refer to the [chat feature guide](/feature-guides/chat) for more information and
instructions.

The `<ChatMessageBell />` requires the [`<Chat/>`](/ui-components/chat) component to be used as well.

## Integration

### `<ChatMessageBell/>`

To show the bell component, you can copy this snippet in your code:

```tsx
<ChatProvider>
    <ChatMessageBell/>
</ChatProvider>
```
