# Chat UI Component

## Introduction

The `<Chat/>` UI component represents the chat message center with sidebar and the chat window. Please refer to
the [chat feature guide](/feature-guides/chat) for more information and instructions.

Alternatively to the `<Chat/>` component, you can also use the individuals parts: `<ChatWindow/>` and 
`<ChatSidebar/>`, see integration guidelines below.

## Integration

### `<Chat/>`

You can place the entire chat on your page using this snippet:

<Tabs items={['React']}><Tab>

```tsx
<ChatProvider>
    <Chat />
<ChatProvider>
```

</Tab></Tabs>

### `<ChatWindow/>`

In case you only want to show the window with all the current chat, then use this snippet: 

<Tabs items={['React']}><Tab>

```tsx
<ChatProvider>
    <ChatWindow />
<ChatProvider>
```

</Tab></Tabs>

### `<ChatSidebar/>`

You can show just the sidebar by using this snippet:

<Tabs items={['React']}><Tab>

```tsx
<ChatProvider>
    <ChatSidebar />
<ChatProvider>
```

</Tab></Tabs>
