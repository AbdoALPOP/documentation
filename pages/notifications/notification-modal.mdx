# Notification Modal UI component

## Introduction

The `<NotificationModal/>` component shows the user's notifications in real-time and also enables the user to
configure the preferences. Please refer to the [notifications feature guide](/notifications) for more
information and instructions.

### `<NotificationModal/>`

{/* TODO Props are not aligned with the actual UI component. */}

The notification modal component can be integrated with just one line of code. The component will open a socket
connection to ROQ Platform to receive notifications events in real time. The list of notifcations is fetched using the
[notifications() API](/notifications/api#notifications).

<Tabs items={['React']}><Tab>

```tsx
import {NotificationModal} from '@roq/ui-react';

function App() {
    return (
        <NotificationModal/>
    );
}
```

#### `<NotificationModal/>` Props list

| Prop    | Type                             | Description                              | Default         |
|---------|----------------------------------|------------------------------------------|-----------------|
| panel   | "notifications" or "preferences" | Controls which view to show in the modal | "notifications" |
| onClose | function                         | Callback, fired when the modal is closed | null            |
