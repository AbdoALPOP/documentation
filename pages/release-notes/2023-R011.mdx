# Project Access Management

<Callout type="info" emoji="ℹ️">

</Callout>

{/* THIS PAGE ISN'T FINAL!!! */}

## Highlights

- Manage your permissions on your project by using ROQ's new project access management capability.
- Hide pages, API routes, and UI elements based on user's permissions
- Filter data based on roles
- Manage your users' access via ROQ Console
