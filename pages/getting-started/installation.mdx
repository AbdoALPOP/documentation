# Local installation

ROQ enables you to generate an entire SaaS application. You can access the complete source code using a GitHub link 
sent to you after using the [AI Assistant](https://app.roq.ai/). The source code is provided to you under MIT license, which 
means you can modify and use it for any purpose. 

A list of technologies is available [here](/getting-started/technology-stack).

  You can find detailed installation instructions in the **README.md** file of your freshly generated application.
