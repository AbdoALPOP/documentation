# Get data of current session

## Client-side

If you want to access the current user's data, e.g., to show the user's first name, you can either use the
useSession hook or an async helper.

## Server-side

You can access the current user's data using this request handler:
<Tabs items={['Next.js']}><Tab>

```tsx
import {getServerSession} from "@roq/nextjs";

async function handler(req: NextApiRequest, res: NextApiResponse) {
    const session = getServerSession(req);
    const user = session.user;
}
```
