# User Account Dropdown

## Introduction

The `<UserAccountDropdown/>` component can be used to show the user's avatar and their account dropdown.
It works with ROQ Authentication, and without.

## Integration

<DoBasicSetup />

### `<UserAccountDropdown/>` with ROQ Authentication

To show the UserAccountDropdown, add this snippet to your page:

<Tabs items={['Reactjs', 'Next.js']}><Tab>

```tsx
import { UserAccountDropdown } from "@roq/ui-react";

function App() {
  return <UserAccountDropdown />;
}
```

</Tab>
<Tab>

```tsx
import { UserAccountDropdown } from "@roq/nextjs";

function App() {
  return <UserAccountDropdown />;
}
```

</Tab>
</Tabs>

### `<UserAccountDropdown/>` without ROQ Authentication

<Tabs items={['Reactjs', 'Next.js']}><Tab>

```tsx
import { UserAccountDropdown } from "@roq/ui-react";

function App() {
  const handleSignOut = () => {
    //your sign-out logic
  };

  return <UserAccountDropdown onSignOut={handleSignOut} />;
}
```
