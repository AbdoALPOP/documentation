# Fetch notifications API

The notification list UI component fetches notifications using the notifications() query. See full specs in the API doc.
This endpoint is available at the client- and server-side APIs.

See the API doc of [fetch notifications](https://snapshotp.roq-platform.com/docs/#query-notifications)

```graphql
query notifications(page: 0, seen: true) {
    totalCount
    data {
      {
        "id": "xyz789",
        "createdAt": "2007-12-03",
        "templateIdentifier": "abc123",
        "content": "xyz789",
        "seen": true,
        "read": false,
        "email": "abc123",
        "providerId": "abc123",
        "title": "abc123",
        "icon": "abc123",
        "lastSeenDate": "xyz789",
        "errorId": "xyz789",
        "errorText": "abc123",
        "status": "sent",
        "channel": "email"
      }
    }
}
```
