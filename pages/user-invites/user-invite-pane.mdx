## Integration

<DoBasicSetup />

You can integrate the `<UserInvitePane/>` component using this code:

```tsx
import { UserInvitePane } from '@roq/ui-react';

function App() {
    return (
        <UserInvitePane
            onInviteError={function noRefCheck(){}}
            onInviteSuccess={function noRefCheck(){}}
        />
    );
}
```
